## 希望实现的算法

* [ ] 单源最短路
* [ ] 中心度
  - [ ] 介数中心度
    出现在其他所有点对最短路上的总次数（相同点对、不同最短路算多次）
    也可以标准化至 [0, 1]
  - [ ] 紧密中心度
    由于是有向图，采用扩展定义：到所有点的最短路的 -α 次方之和的 α 次方
* [ ] 连通分量（有向图，所以是强连通分量）
* [ ] PageRank
